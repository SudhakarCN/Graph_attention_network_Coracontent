# Graph_attention_network_Coracontent
pandas, numpy, networkx, stellargraph, tensorflow, keras, sklearn, matplotlib

Language and version: Python 3.8
IDE: Jupyter notebook
Environment: Ubuntu 18.04

Instructions:
1. Install all the required libraries.
2. Two jupyter notebooks are presented.
	1. task_1.ipynb
	2. task_2.ipynb
3. Open the notebook. (example: task_1.ipynb)
4. Each cell is executable. You can either execute cell by cell or all cells can be executed at once.
5. To execute all the cells at once
	1. Open the notebook
	2. Click on Cell option in the menu
	3. Click on Run all option.
